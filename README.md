# Discover India — InterYouth Fest 2026

Interactive web page representing India at Istinye University's InterYouth Festival (May 8, 2026).

**Live:** https://india-interyouth.onrender.com

## Features

- **45+ fact-checked cards** with thumbnail images across 8 categories (Science, Math, Language, Culture, Food, Geography, Cinema, Records)
- **16-slide photo gallery** with pause/play button — images from *Indian Contributions to Science* (Vijnana Bharati) and *Destination India* (Embassy of India, Vienna)
- **Spin the Chakra quiz** — interactive wheel game with 12 questions, confetti on correct answers
- **Hindi instrumental BGM** — auto-plays on interaction, mute toggle in bottom-right
- **India flag-themed UI** — saffron, white, navy (Ashoka Chakra), green color zones with smooth gradient transitions
- **Mobile-first** — designed for QR code scanning at the booth
- **Favicon** — India flag icon in browser tab
- **Istinye University logo** in hero section

## Fact-Checking

All claims were reviewed against primary sources (ISRO, UNESCO, Census of India, Encyclopaedia Britannica, Vijnana Bharati). Viral overclaims were removed or reworded. Full source list in `india-fact-sources.txt`.
